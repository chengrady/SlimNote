# SlimNote - Typora风格编辑器使用指南

## 🎨 新功能概述

SlimNote 现在支持类似Typora的**所见即所得(WYSIWYG)** Markdown编辑体验！

## ✨ 核心特性

### 1. 双模式编辑

- **✏️ 所见即所得模式** (默认)
  - 实时渲染Markdown
  - 专注于内容创作
  - 隐藏Markdown语法

- **📝 源码模式**
  - 传统编辑器体验
  - 支持分屏预览
  - 可查看原始Markdown

**切换方式**：点击工具栏的 ✏️ 或 📝 按钮

### 2. 快捷键支持

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+B` / `Cmd+B` | **加粗** |
| `Ctrl+I` / `Cmd+I` | *斜体* |
| `Ctrl+K` / `Cmd+K` | [链接](url) |
| `Ctrl+Shift+C` | 代码块 |
| `Ctrl+Shift+X` | 删除线 |
| `Tab` | 增加列表缩进 |
| `Shift+Tab` | 减少列表缩进 |

### 3. 图片支持

#### 拖放插入
直接将图片文件拖拽到编辑器中即可插入

#### 粘贴插入
- 截图后直接 `Ctrl+V` 粘贴
- 从剪贴板粘贴图片

图片会自动转换为 base64 格式嵌入到文档中

### 4. 数学公式

支持 KaTeX 数学公式渲染

**行内公式**：
```markdown
质能方程: $E = mc^2$
```

**块级公式**：
```markdown
$$
\sum_{i=1}^{n} i = \frac{n(n+1)}{2}
$$
```

### 5. 表格编辑

支持 GitHub Flavored Markdown (GFM) 表格

**快速插入**：点击工具栏的表格按钮，自动插入表格模板

```markdown
| Header 1 | Header 2 |
| -------- | -------- |
| Cell 1   | Cell 2   |
```

### 6. 代码高亮

支持多种编程语言的语法高亮：

````markdown
```javascript
function hello() {
  console.log("Hello, World!");
}
```
````

支持的语言：JavaScript, Python, CSS, HTML, JSON, 等

### 7. 其他Markdown特性

#### 任务列表
```markdown
- [ ] 待办事项
- [x] 已完成事项
```

#### 引用块
```markdown
> 这是一段引用文本
```

#### 分割线
```markdown
---
```

#### 列表
```markdown
- 无序列表项1
- 无序列表项2

1. 有序列表项1
2. 有序列表项2
```

## 🎯 使用技巧

### 快速开始编辑
1. 打开任意 `.md` 文件
2. 默认进入所见即所得模式
3. 直接开始输入内容

### 切换到源码模式
1. 点击工具栏的 📝 按钮
2. 可以看到原始的Markdown语法
3. 点击 👁️ 按钮启用分屏预览

### 插入特殊内容
1. **加粗文字**：选中文字后按 `Ctrl+B`
2. **插入链接**：按 `Ctrl+K`，输入URL
3. **插入图片**：拖放图片文件到编辑器
4. **插入表格**：点击工具栏表格按钮

## 🎨 主题适配

编辑器会自动适配你选择的主题：

- **浅色主题**：清爽明亮的编辑体验
- **深色主题**：护眼的深色编辑界面

在 设置 →主题 中切换主题，编辑器会自动更新

## ⚙️ 设置选项

编辑器支持以下设置：

- **字体大小**：自定义编辑器文字大小
- **自动保存**：编辑内容自动保存
- **主题切换**：浅色/深色主题

## 🐛 常见问题

### Q: 如何查看Markdown源码？
A: 点击工具栏的 📝 按钮切换到源码模式

### Q: 图片无法插入？
A: 确保图片格式为常见格式（JPG, PNG, GIF, WebP）

### Q: 数学公式不显示？
A: 确保使用正确的LaTeX语法，行内公式用 `$...$`，块级公式用 `$$...$$`

### Q: 表格格式错乱？
A: 表格需要遵循GFM规范，确保每行有相同数量的列

### Q: 快捷键不起作用？
A: 确保使用英文输入法，某些快捷键可能与系统快捷键冲突

## 🚀 性能优化

- **大型文档**：对于超大文档，建议使用源码模式以获得更好性能
- **图片处理**：图片会转换为base64，大量图片会增加文件大小
- **自动保存**：编辑内容会实时同步到store
